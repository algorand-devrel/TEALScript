> **bzero**\<`T`\>(`size`?): `T`

Returns zero bytes of the given size.

• **T** = [`Brand`](../type-aliases/Brand.md)\<`string`, `"bytes"`\>

## Parameters

• **size?**: [`IntLike`](../type-aliases/IntLike.md)

the number of zero bytes to return. If not given, returns the size of the type given
the type argument
